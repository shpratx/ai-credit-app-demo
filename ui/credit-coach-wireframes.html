Design System References

  • Colours: Lloyds Green (#006A4D), surfaces, semantic status palette
  • Typography: GT Ultra Standard/Medium Bold, type scale per design system
  • Components: Cards (12px radius), Buttons (8px radius, 52px height), Inputs, Badges, Alerts, Chat bubbles
  • Spacing: 4px base grid (16px card padding, 24px section gaps)
  • Accessibility: WCAG 2.1 AA, 48dp touch targets, focus rings, screen reader labels
  • Chat UI: Section 5.18 of Lloyds Design System (message bubbles, quick replies, input bar)

Regulatory Notes

  • Consent screen: GDPR Art. 6/7 (explicit consent), UK GDPR Art. 13 (transparency at collection)
  • Score display: FCA Consumer Duty (consumer understanding outcome)
  • Soft search: CONC 5.2A (no credit footprint for monitoring)
  • Data retention: FCA SYSC 9.1.1R (6-year retention noted in consent settings)
  • Estimates disclaimer: PRA FS2/23 (explainability — label as estimates)
