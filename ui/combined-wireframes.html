Design System References

  • Colours: Lloyds Green (#006A4D), surfaces, semantic status palette
  • Typography: GT Ultra Standard/Medium Bold, type scale per design system
  • Components: Cards (12px radius), Buttons (8px radius, 52px height), Inputs, Badges, Alerts, Chat bubbles
  • Spacing: 4px base grid (16px card padding, 24px section gaps)
  • Accessibility: WCAG 2.1 AA, 48dp touch targets, focus rings, screen reader labels
  • Chat UI: Section 5.18 of Lloyds Design System (message bubbles, quick replies, input bar)

Regulatory Notes

  • Consent screen: GDPR Art. 6/7 (explicit consent), UK GDPR Art. 13 (transparency at collection)
  • Score display: FCA Consumer Duty (consumer understanding outcome)
  • Soft search: CONC 5.2A (no credit footprint for monitoring)
  • Data retention: FCA SYSC 9.1.1R (6-year retention noted in consent settings)
  • Estimates disclaimer: PRA FS2/23 (explainability — label as estimates)

Design Decisions

  • Explanation per action ("Why"): PRA FS2/23 requires explainability for AI recommendations. Each action includes reasoning.
  • Disclaimer on plan: FCA Consumer Duty — estimates must be clearly labelled as such.
  • Disputed account badge: Domain KB PD24 — customers can dispute CRA data. UI must show dispute status clearly.
  • Debt-to-income ratio: Domain KB PD3 — DBR is a key affordability metric. Shown with band classification.
  • Spending impact from FA: Product Vision Section 3.2 — augments existing FA spending insights with credit context.
  • Chart text alternative: WCAG 2.1 AA — charts must have text descriptions for screen readers.
  • Milestone modal: Design System Section 5.14 — uses modal pattern with focus trap and Escape dismiss.

Regulatory Compliance Notes

  • CONC 3.5.5R: Representative example with all 6 values (amount, term, rate, APR, monthly, total) shown on offer screen
  • CONC 3.5.3R: Representative APR is the most prominent rate figure on the offer card
  • CCA s.55A: SECCI screen provides full pre-contract credit information before acceptance
  • CCA s.66A: 14-day cooling-off period clearly communicated on offer and SECCI screens
  • CCA s.94: Right to early repayment (0% fee) disclosed in SECCI
  • CONC 7.7.5R: Late payment fee capped at £12 — disclosed in SECCI
  • CONC 5.2A: Affordability assessment completed before offer generation (not shown to customer but referenced in audit trail)
  • CONC 7.3: Offers suppressed during financial distress — debt advice signposted (StepChange, Citizens Advice, National Debtline)
  • FCA Consumer Duty: Consumer understanding outcome — plain language, total cost prominent, no jargon
  • PRA SS1/23: Simulation model confidence level disclosed; estimates clearly labelled
  • PRA FS2/23: Simulation logic explainable — "How this works" section shows factor changes
